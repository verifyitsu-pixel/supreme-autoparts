# Supreme Autoparts WordPress Store

`supreme-autoparts-theme` is an installable WooCommerce-ready theme matching the rebuilt storefront brand.

To publish it, create a zip of the theme folder and upload it in **WordPress → Appearance → Themes → Add New**, activate it, install WooCommerce, then assign the primary navigation menu. Automated remote installation requires the target WordPress URL plus an administrator application password or SSH/SFTP access.
